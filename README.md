# AI Document Assistant

## Python Script / App
Streamlit app that allows a user to upload a document (PDF or TXT) and ask questions about it.  
The AI reads the document and answers questions based only on the uploaded file.

## Model Name & Source
Model: Gemini 3 Flash Preview  
Source: Google Generative AI (Google GenAI API)

## Rationale for Selection
Gemini 3 Flash Preview was selected because it is fast, efficient, and designed for real-time applications. The model works well for chat-based interactions and document question answering. It also integrates easily with Python through the Google GenAI library, making it suitable for a simple Streamlit application that processes uploaded documents and responds quickly to user questions.

## Reflection on Responsible AI Use (100–150 words)
Responsible AI use is important when building applications that generate answers from user data.
In this app, the AI is instructed to only answer questions using the uploaded document and to say it does not know if the information is not present. 
This helps reduce the risk of hallucinations or incorrect information. 
Another responsible practice is being transparent about the model being used so users understand that responses are generated by AI and may not always be perfect.
It is also important to handle uploaded documents carefully and avoid storing sensitive information unnecessarily.
Developers should design systems that prioritize accuracy, transparency, and user privacy when working with AI tools.
